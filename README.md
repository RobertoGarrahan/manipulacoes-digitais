## üé® Concept

The website functions as a **digital maze** made up of a sequence of images.  
Each image contains **specific clickable areas** (image mapping) that redirect the user to other images.

- Clicking the *correct* areas allows the user to progress through the labyrinth.
- Incorrect paths lead to alternative images, creating disorientation and exploration.
- By following the right sequence of interactions, the user eventually reaches the end of the maze.

The project blends **digital navigation** with **physical space**, expanding the experience beyond the screen.

---

## üí° Physical Interaction (Arduino)

In addition to the digital experience, the project integrates with an **Arduino**:

- Certain interactions trigger **physical lights** in a real-world labyrinth.
- This creates a connection between the virtual maze and a physical installation.
- The digital choices directly influence the physical environment.
